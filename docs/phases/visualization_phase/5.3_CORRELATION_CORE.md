## 3. Per-Hypothesis Specification

### 3.1 H1.1 — Global Positive Correlation

**Population figure** (2×3 panels, full width 7.0"): See base document §3.1(A).
- Panels (a–c): Hexbin scatter Lev vs GED for IAM LOW, LINUX, AIDS. OLS + identity lines. ρ, r, n_pairs annotated.
- Panel (d): Horizontal bar of ρ per dataset with BCa 95% CI error bars + significance stars.
- Panel (e): Horizontal bar of Pearson r, same format.
- Panel (f): Grouped bar of P@k for k ∈ {5, 10, 20} by dataset.

**Individual figure** (3 rows × 3 cols): See base document §3.1(B).
- Row 1: Similar concordant pair (GED=1, min Lev). Use `select_concordant_similar()`.
- Row 2: Dissimilar concordant pair (high GED, high Lev). Use `select_dissimilar()`.
- Row 3: Most discordant pair. Use `select_discordant()`.
- Each row: two graphs drawn + alignment strip below.
- Record selections in `SelectedExample` for the manifest.

**Table**: One row per dataset. Columns: Dataset, N, Pairs, ρ [CI], r [CI], τ, CCC, P@10, Mantel p. Best/worst ρ highlighted.

### 3.2 H1.2 — Monotone Degradation

**Population figure** (single-column 3.39"): See base document §3.2(A).
- X = distortion (LOW, MED, HIGH). Y = ρ. Two lines: exhaustive (solid) + greedy (dashed). CI bands. JT test statistic and p-value annotated.

**Individual figure** (3 cols × 2 rows): See base document §3.2(B).
- Pick same letter class (e.g., "A"). One pair per distortion level with GED=1. Show graphs + alignments.

**Table**: 3 rows (LOW/MED/HIGH). Columns: Distortion, ρ_exh [CI], ρ_greedy [CI], Δρ, JT p-value.

### 3.3 H1.3 — Density Stratification

**Population figure** (single-column): See base document §3.3(A).
- Bin pairs into 4–5 density quartiles. Box/violin plot of Lev–GED rank agreement per bin. Overlay: ρ per bin connected by a line with CI bands.

**Individual figure** (2 rows × 2 cols): See base document §3.3(B).
- Top: sparse pair from LINUX (tree-like). Bottom: dense pair from AIDS or IAM HIGH. Both with graphs + alignments.

**Table**: Rows = density bins. Columns: Density bin, N pairs, ρ [CI], Mean |GED − d_L|, Concordance.
