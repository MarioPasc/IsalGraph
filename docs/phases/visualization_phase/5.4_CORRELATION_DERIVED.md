## 2. Per-Hypothesis Specification

### H1.4 — Within-Class vs Between-Class Discrimination

See base document §3.4. Only applies to IAM Letter datasets (3 datasets with class labels).

**Population figure** (1×3 panels or stacked): Split-violin per IAM distortion level. Left = within-class Lev, right = between-class. Annotate Cohen's d and Mann-Whitney U p-value with significance bracket.

**Individual figure**: Top row = two "A" graphs (IAM LOW, small Lev). Bottom row = one "A" + one "M" graph (large Lev). Annotate distances.

**Table**: 3 rows (IAM LOW/MED/HIGH). Columns: d_L within (μ ± σ), d_L between (μ ± σ), Cohen's d, U test p.

### H1.5 — Exhaustive vs Greedy

See base document §3.5.

**Population figure** (full width, 1×2 panels):
- (a) Grouped bar: per dataset, two bars (exhaustive blue, greedy orange) showing ρ ± 95% CI.
- (b) Scatter of ρ_exhaustive vs ρ_greedy across 5 datasets with identity line.

**Individual figure**: Select a graph where exhaustive ≠ greedy from `method_comparison/{dataset}_comparison.json`. Show graph, both strings, and alignment between them. Also show one graph where they agree (majority case).

**Table**: Columns: Dataset, ρ_exh [CI], ρ_greedy [CI], Δρ [CI], Agreement %.
