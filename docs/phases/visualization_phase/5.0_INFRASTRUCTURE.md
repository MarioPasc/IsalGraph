## 2. Module Specifications

### 2.1 `result_loader.py` (~150 lines)

**Purpose**: Load every JSON, CSV, and NPZ file from Agents 0–4 into typed dataclasses.

**Key design decisions**:
- Use `@dataclass` for typed results (see base document §4.2 for the full dataclass hierarchy: `DatasetArtifacts`, `CorrelationStats`, `EmbeddingStats`, `ComputationalStats`, `EncodingStats`, `AllResults`).
- Graceful degradation: if an agent's outputs are missing (e.g., Agent 3 hasn't run yet), return `None` for that section rather than crashing. This lets Phase 1 agents run even if not all prior agents have completed.
- All paths are relative to configurable root directories (no hardcoded paths).
- Load NPZ matrices with `allow_pickle=True` for metadata dicts.
- Load JSON with `encoding="utf-8"`.

**Functions to implement**:
```python
def load_all_results(data_root, correlation_dir, embedding_dir, computational_dir, encoding_dir) -> AllResults
def _load_dataset_artifacts(data_root, dataset) -> DatasetArtifacts
def _load_correlation_stats(correlation_dir, dataset, method) -> CorrelationStats | None
def _load_embedding_stats(embedding_dir, dataset, method) -> EmbeddingStats | None
def _load_computational_stats(computational_dir, dataset) -> ComputationalStats | None
def _load_encoding_stats(encoding_dir) -> EncodingStats | None
```

**Constants**:
```python
ALL_DATASETS = ["iam_letter_low", "iam_letter_med", "iam_letter_high", "linux", "aids"]
METHODS = ["exhaustive", "greedy"]
LABELED_DATASETS = {"iam_letter_low", "iam_letter_med", "iam_letter_high"}

    "iam_letter_low": "IAM Letter LOW", "iam_letter_med": "IAM Letter MED",

}
```

### 2.2 `table_generator.py` (~100 lines)

**Purpose**: Generate paired LaTeX tables (color + grayscale) with `\best{}`/`\worst{}` highlighting and significance markers.

**Key design**: See base document §4.3 for the full specification. The core function `generate_dual_table()` takes a DataFrame and outputs two `.tex` files. Use `booktabs` formatting. The `_highlight_best_worst()` function applies per-column highlighting. The `_format_significance()` function maps p-values to `***`/`**`/`*`/`n.s.`.

**Functions to implement**:
```python
def generate_dual_table(df, output_dir, basename, caption, label, highlight_cols, note) -> tuple[Path, Path]
def format_significance(p: float) -> str
```

### 2.3 `graph_drawing.py` (~80 lines)

**Purpose**: Draw NetworkX graphs with publication-quality styling.

**Key design**: See base document §4.5. Uses `nx.spring_layout(seed=42)`. Paul Tol colors. Node labels in white on colored circles. Supports edge highlighting (for showing edited edges). Canonical string `w*` displayed below in monospace.

**Functions to implement**:
```python
def draw_graph(G, ax, title, canonical_string, highlight_edges, highlight_color, node_color, edge_color, node_size, layout_seed) -> None
```

### 2.4 `string_alignment.py` (~100 lines)

**Purpose**: Compute Levenshtein alignment via DP backtrace and render as colored two-row character grid.

**Key design**: See base document §4.6. The DP backtrace returns a list of `(operation, char_s, char_t)` tuples. The rendering uses `FancyBboxPatch` for each character cell, color-coded by operation type (match=green, substitute=red, insert=blue, delete=yellow, all Paul Tol).

**Functions to implement**:
```python
def levenshtein_alignment(s, t) -> list[tuple[str, str | None, str | None]]
def draw_alignment(alignment, ax, s_label, t_label) -> None
```

### 2.5 `cdll_drawing.py` (~80 lines)

**Purpose**: Draw CDLL as a circular ring with pointer arrows. Used by the algorithm illustration figures (5.1).

**Key design**: See addendum §B.6. Nodes placed evenly around a circle. Primary pointer (π) as red arrow, secondary (σ) as blue. New nodes highlighted in yellow. Handles π = σ overlap by offsetting arrows.

**Functions to implement**:
```python
def draw_cdll_ring(ax, cdll_order, primary_ptr_idx, secondary_ptr_idx, new_node_payload, radius) -> None
```

### 2.6 `example_selector.py` (~120 lines)

**Purpose**: Deterministic selection of illustrative graph pairs for individual-level figures. All selections are logged to `selected_examples.json`.

**Key design**: See base document §4.4. Selection strategies: `select_concordant_similar` (small GED + small Lev, same class preferred), `select_dissimilar` (large GED + large Lev, different class), `select_discordant` (max rank disagreement), `select_knn_example` (high or low P@k).

**Functions to implement**:
```python
def select_concordant_similar(ged, lev, node_counts, labels, target_ged, min_nodes, max_nodes) -> tuple[int, int]
def select_dissimilar(ged, lev, node_counts, labels) -> tuple[int, int]
def select_discordant(ged, lev) -> tuple[int, int]
def select_knn_example(ged, lev, k, mode) -> int   # mode="best" or "worst"
def save_selection_log(selections, output_path) -> None
```

**Dataclass**:
```python

    role: str

    indices: tuple[int, ...]
    graph_ids: tuple[str, ...]
    criterion: str
```

## 3. Testing Plan

### 3.1 Unit tests for each module

```python
# test_result_loader.py
def test_load_missing_agent_graceful():
    """Loading with missing agent dir returns None, not crash."""
    results = load_all_results("data/eval", "/nonexistent", "/nonexistent", "/nonexistent", "/nonexistent")
    assert results.correlation == {}

def test_load_single_dataset():
    """Load one dataset's artifacts and verify shapes."""
    arts = _load_dataset_artifacts("data/eval", "iam_letter_low")
    assert arts.ged_matrix.shape[0] == arts.ged_matrix.shape[1]
    assert len(arts.graph_ids) == arts.ged_matrix.shape[0]

# test_table_generator.py
def test_generate_dual_table(tmp_path):
    """Generate both color and gray tables."""
    df = pd.DataFrame({"A": ["x", "y"], "B": [0.5, 0.3]})
    c, g = generate_dual_table(df, tmp_path, "test", "cap", "lbl", {"B": True})
    assert c.exists() and g.exists()
    assert "\\best" in c.read_text()

# test_graph_drawing.py
def test_draw_graph_runs():
    """draw_graph produces a figure without error."""
    G = nx.cycle_graph(4)
    fig, ax = plt.subplots()
    draw_graph(G, ax, title="C4", canonical_string="VNVNC")
    fig.savefig("/tmp/test_draw.png")
    assert os.path.exists("/tmp/test_draw.png")

# test_string_alignment.py
def test_alignment_identity():
    """Identical strings produce all matches."""
    al = levenshtein_alignment("VNC", "VNC")
    assert all(op == "match" for op, _, _ in al)

def test_alignment_insert():
    """Insertion detected correctly."""
    al = levenshtein_alignment("VC", "VNC")
    ops = [op for op, _, _ in al]
    assert "insert" in ops

# test_cdll_drawing.py
def test_cdll_ring_draws():
    """draw_cdll_ring runs without error."""
    fig, ax = plt.subplots()
    draw_cdll_ring(ax, [0, 1, 2], primary_ptr_idx=0, secondary_ptr_idx=1, new_node_payload=2)
    fig.savefig("/tmp/test_cdll.png")

# test_example_selector.py
def test_select_concordant():
    """Selects a pair with GED=1."""
    ged = np.array([[0, 1, 5], [1, 0, 4], [5, 4, 0]])
    lev = np.array([[0, 2, 8], [2, 0, 7], [8, 7, 0]])
    nc = np.array([5, 6, 7])
    pair = select_concordant_similar(ged, lev, nc, target_ged=1, min_nodes=3, max_nodes=10)
    assert pair == (0, 1)
```
