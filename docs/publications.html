2026 arXiv preprint

Instruction set for the representation of graphs

Ezequiel Lopez-Rubio, Mario Pascual-Gonzalez

This work introduces IsalGraph, a framework for representing the structure of finite simple graphs as strings over a 9-character instruction alphabet. Using a circular doubly linked list (CDLL) with two mobile pointers, any graph can be encoded into a compact instruction sequence and faithfully decoded back. We prove that the canonical string — the lexicographically smallest shortest encoding over all starting vertices — is a complete graph invariant: two graphs are isomorphic if and only if their canonical strings coincide. The Levenshtein distance between canonical strings defines a metric on isomorphism classes that serves as a polynomial-time proxy for graph edit distance.

2025 arXiv preprint

Representation of the structure of graphs by sequences of instructions

Ezequiel Lopez-Rubio

This paper presents the foundational theory for representing graph structures using instruction sequences. A graph is encoded as a string over a finite alphabet by traversing it with a circular doubly linked list equipped with two pointers. The encoding procedure is reversible: the original graph can be reconstructed from its string representation. This framework opens new avenues for graph comparison, classification, and generation using techniques from string processing.

2025 Journal

Representation of Molecules by Sequences of Instructions

Karl Thurnhofer-Hemsi, Iván García-Aguilar, José David Fernández-Rodriguez, Ezequiel López-Rubio

This paper introduces IsalChem, the predecessor to IsalGraph, which applies instruction string encoding to molecular graphs. Molecules are represented as labeled graphs where atoms are nodes and bonds are edges. The encoding framework produces compact string representations that capture molecular topology, enabling string-based techniques for molecular comparison, generation, and classification. Published in the Journal of Chemical Information and Modeling (ACS Publications).
